# menix

A minimal and expandable Unix-like operating system.

This is a hobby project to learn OS development.
The kernel itself uses a hybrid microkernel architecture.
It handles:
- Booting
- Interprocess Communication
- Scheduling
- Virtual File System
- Virtual Memory
